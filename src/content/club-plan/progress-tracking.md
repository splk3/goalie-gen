## Progress Tracking & Evaluation

Goalie progress is tracked using coach observation, goalie-specific skill benchmarks, and player feedback.

Formal goalie evaluations are conducted [GOALIE_EVALUATIONS_WHEN].

Each evaluation cycle includes:

- technical assessment (movement, save execution, rebound control, puck tracking)
- tactical assessment (positioning, game-reading, decision-making)
- mental and leadership assessment (composure, communication, resilience)
- written feedback with action items for the next training block
