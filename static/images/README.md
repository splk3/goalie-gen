# Logo Assets

This directory contains the Goalie Gen logos for use throughout the site.

## Required Files

Please download and place the following files from the GitHub issue:

### Full Logos
- `logo-dark.png` - Dark mode full logo (goalie in net with "GOALIE GEN DEVELOPMENT PLANS" text) - white on transparent
- `logo-light.png` - Light mode full logo (goalie in net with "GOALIE GEN DEVELOPMENT PLANS" text) - blue/red colors

### Alternate Logos  
- `logo-alt-dark.png` - Dark mode alternate logo (gear with "GG" letters) - white on transparent
- `logo-alt-light.png` - Light mode alternate logo (gear with "GG" letters) - red/blue colors
