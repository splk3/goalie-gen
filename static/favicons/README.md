# Favicon Assets

This directory contains favicon files for the Goalie Gen site.

## Required Files

Please download and extract the following zip files from the GitHub issue:

### Light Mode Favicons
Download: https://github.com/user-attachments/files/24362453/gg-favicon.zip

Extract all files to this directory with `-light` suffix:
- `favicon-light.ico`
- `favicon-16x16-light.png`
- `favicon-32x32-light.png`
- `apple-touch-icon-light.png`
- `android-chrome-192x192-light.png`
- `android-chrome-512x512-light.png`
- etc.

### Dark Mode Favicons
Download: https://github.com/user-attachments/files/24362454/gg-favicon-darkmode.zip

Extract all files to this directory with `-dark` suffix:
- `favicon-dark.ico`
- `favicon-16x16-dark.png`
- `favicon-32x32-dark.png`
- `apple-touch-icon-dark.png`
- `android-chrome-192x192-dark.png`
- `android-chrome-512x512-dark.png`
- etc.

## Note

The site will use light mode favicons by default. Dark mode favicon support requires additional browser configuration.
